# Cheatsheet

## Navigation

- **M-s-right**: Next window
- **M-s-left**: Previous window

## Managing text

- **M-down**: Move current line down
- **M-up**: Move current line up

## Finding files

- **C-p** : Find files in project

## Git

- **F6**: Git status
